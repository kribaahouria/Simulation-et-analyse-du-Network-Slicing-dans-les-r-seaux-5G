# SliceSim 5G - Simulation Network Slicing

## Description
Projet Python simulant le **Network Slicing** dans un réseau **5G**.  
Trois slices simulées :  
- **eMBB** (enhanced Mobile Broadband)  
- **URLLC** (Ultra-Reliable Low Latency Communication)  
- **mMTC** (Massive Machine-Type Communications)

La simulation utilise **SimPy** pour générer du trafic et mesurer les délais moyens.
